# Hello Mission

The smallest possible mission. Demonstrates the `MissionBase` lifecycle: `Initialize` → `Begin` → `CompleteMission`.

The sample includes a single scene with one GameObject carrying a `HelloMission` component. On `Begin`, the mission waits 5 seconds and then reports a fixed score.

Use this sample to verify your project is wired up correctly before writing your own mission.

> Scaffold lands during M1.
